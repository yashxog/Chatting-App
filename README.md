## Installation

Client - front end part of the app
Server - Back end part of the app

CLIENT:-
```bash
  npm install
  npm start
  npm install axios
```

SERVER:-
```bash
  npm init -y 
  npm install bcrypt crypto dotenv express getstream stream-chat nodemon twilio 
  npm start

```
